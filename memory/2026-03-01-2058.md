what is export command and should I do something except it? how can I know it has worked
assistant: **`export` sets an environment variable** — it's a shell command that makes a key available to all programs started from that terminal session.

**To set the key:**

```bash
export GEMINI_API_KEY="your-actual-key-here"
```

**What you need to know:**

1. **It only lasts for the current terminal session** — close the terminal and it's gone
2. **To make it permanent**, add it to your shell config file:
   - For bash: `echo 'export GEMINI_API_KEY="your-key"' >> ~/.bashrc`
   - For zsh: `echo 'export GEMINI_API_KEY="your-key"' >> ~/.zshrc`
   - Then run `source ~/.bashrc` (or `~/.zshrc`) to reload

**How to verify it worked:**

```bash
echo $GEMINI_API_KEY
```

This should print your key. If it shows nothing, the variable isn't set.

After that, run:

```bash
openclaw models set google/gemini-2.0-flash
```

And you're good.
